" let Vundle manage Vundle, required

" Enable Powerline

"split navigations
nnoremap <C-J> <C-W><C-J>
nnoremap <C-K> <C-W><C-K>
nnoremap <C-L> <C-W><C-L>

" Enable folding with the spacebar

"To add proper PEP 8 indentation.
au BufNewFile,BufRead *.py
    \ set tabstop=4 |
    \ set softtabstop=4 |
    \ set shiftwidth=4 |
    \ set textwidth=79 |
    \ set expandtab |
    \ set autoindent |
    \ set fileformat=unix

" For full stack development, you can use another au:
au BufNewFile,BufRead *.js, *.html, *.css
    \ set tabstop=2 |
    \ set softtabstop=2 |
    \ set shiftwidth=2 

" Flagging Unnecesary Whitespace
highlight BadWhitespace ctermbg=red guibg=darkred
au BufRead,BufNewFile *.py,*.pyw,*.c,*.h match BadWhitespace /\s\+$/

"UTF-8 Support
set encoding=utf-8


" Make code look pretty:

" Define color scheme based on VIM mode:

  set background=dark
  colorscheme solarized
else
  colorscheme zenburn
endif

" Solarized also ships with a dark and light theme. To make switching
" between them very easy (by pressing F5):
call togglebg#map("<F5>")
